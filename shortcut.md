# ⌨️ میانبرهای صفحه کلید و تنظیمات VS Code

## 🪟 میانبرهای عمومی ویندوز

| میانبر             | کاربرد                                                                 |
| ------------------ | ---------------------------------------------------------------------- |
| `Win + D`          | کوچک کردن همه پنجره‌ها (نمایش دسکتاپ)                                  |
| `Win + V`          | نمایش تاریخچه کلیپ‌بورد (در صورت فعال بودن)                            |
| `Win + .`          | باز کردن صفحه ایموجی‌های ویندوز                                        |
| `Ctrl + T`         | تب جدید در مرورگر                                                      |
| `Ctrl + W`         | بستن تب فعلی                                                           |
| `Ctrl + Tab`       | جابه‌جایی بین تب‌های یک برنامه                                         |
| `Ctrl + Shift + L` | اگر کلمه‌ای انتخاب شده باشد، همه کلمه‌های مشابه در صفحه انتخاب می‌شوند |
| `Ctrl + D`         | یکی یکی انتخاب کردن کلمه‌های مشابه (Multiple cursors)                  |
| `F5`               | رفرش کامل صفحه                                                         |
| `Shift + Delete`   | حذف کامل خط جاری + کپی شدن خودکار آن                                   |

## 🎨 میانبرهای متنی (در ویرایشگرها)

| میانبر                        | کاربرد                                             |
| ----------------------------- | -------------------------------------------------- |
| `Ctrl + /`                    | کامنت کردن خط یا انتخاب                            |
| `Ctrl + S`                    | ذخیره فایل (همراه با اعمال Prettier در صورت تنظیم) |
| `Ctrl + Z` / `Ctrl + Y`       | عقب / جلو بردن تغییرات                             |
| `Ctrl + +` / `Ctrl + -`       | بزرگ/کوچک کردن صفحه (زوم)                          |
| `Alt + نگه داشتن + کلیک راست` | انتخاب چند ناحیه متن به صورت ستونی                 |

## ⚙️ تنظیمات مهم VS Code

| تنظیمات                               | مسیر / روش                                                         |
| ------------------------------------- | ------------------------------------------------------------------ |
| باز کردن `settings.json`              | `Ctrl + Shift + P` → `Preferences: Open User Settings (JSON)`      |
| نمایش همه شورت‌کات‌ها                 | `Ctrl + K + S`                                                     |
| Auto Save هنگام تغییر تب              | `settings` → `auto save` → انتخاب `onFocusChange`                  |
| تنظیم Prettier به عنوان فرمتر پیش‌فرض | `settings` → `default format` → انتخاب `Prettier - Code formatter` |
| فعال کردن format on save              | `settings` → `format on save` → تیک بزنید                          |
| فعال کردن ESLint خودکار               | `settings` → `eslint run` → تیک بزنید                              |
